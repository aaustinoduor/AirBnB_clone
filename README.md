# 0x00 AirBnB clone

This is the console version ofthe AirBnB Clone project

## 1.0 Start the console app

```Bash
   ./console.py

```

## 2.0 How to use

### 2.1 Show models

```Bash
    (hbnb) all MyModel

```

### 2.2 Show a model

```pwsh
    (hbnb) show BaseModel [model_id]

```

### 2.3 Create a model

```bash
    (hbnb) create BaseModel

```

### 2.4 Delete a model

```bash
    destroy BaseModel [model_id]

```
